# Summary

These are DXF and CDR (Corel Draw) files for the laser-cut acrylic chassis I used in my [line-following robot, Usain Volt 2.0](https://www.pololu.com/blog/485/jons-line-following-robot-usain-volt-2.0), for the [2015 Las Vegas robotics club line following competition](https://www.pololu.com/blog/493/video-lvbots-april-2015-line-following-competition).

I didn't take any pictures of the pieces before (or during) assembly, so unfortunately all of the visuals I have available are of the finished robot.

160214 EDIT: Thingiverse user, [Melodious Zero](http://www.thingiverse.com/MelodiousZero/about), was able to use these DXF files as a template for a 3D model for another line follower robot.  I've updated this page to include his STL file, as well as a few pictures from his 3D print.  The STL should have the following specifications:

Height: 4 mm
Length: 182.21 mm 
Width: 108.5 mm
